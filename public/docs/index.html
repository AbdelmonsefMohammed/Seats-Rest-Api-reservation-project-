    Introduction

    This documentation aims to provide all the information you need to work with My Seats API.

    As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

    Base URL

    http://localhost

    Authenticating requests

    This API is not authenticated.

    Endpoints

    api/v1/getbranches/{category}

    Example request:
    
curl --request GET \
    --get "http://localhost/api/v1/getbranches/all?distance=20" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Get: Branches in home screen endpoint"```
    const url = new URL(
    "http://localhost/api/v1/getbranches/all"
);

let params = {
    "distance": "20",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
    "Get": "Branches in home screen endpoint",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

    Request      

    GET api/v1/getbranches/{category}

    URL Parameters

    category  string optional  

    Query Parameters

    lat  string optional  

    you send the lat of the user (User need to enable GPS).

    lng  string optional  

    you send the lng of the user (User need to enable GPS).

    distance  string optional  

    you send the max distance you want to search for By (km) the logged in user By dafault the distance is 20 km (can only be used if lat and lng are provided) .
